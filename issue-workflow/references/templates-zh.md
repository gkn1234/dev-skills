# Issue 模板 - 中文

## User Story 模板

```markdown
## 用户故事
作为一个【角色】，我想要【功能】，以便【价值】。

## Design
📄 （待创建）

## 验收标准
（待创建）
```

## Design 模板

```markdown
## 概述
【设计文档内容摘要】

## 设计文档
📄 [【文档路径】](【文档链接】)

## User Story
📄 #【编号】

## Tasks
（待创建）
```

## Task 模板

```markdown
## 任务描述
【任务目标简述】

📄 #【编号】

```

## Test Cases 模板

```markdown
## 测试用例
- [ ] 【用例描述】

📄 #【编号】

## Task
📄 #【编号】
```

## 标签

| 类型 | 标签名 | 颜色 |
|------|--------|------|
| User Story | `user-story` | `0052CC` (蓝色) |
| Design | `design` | `0E8A16` (绿色) |
| Task | `task` | `FBCA04` (黄色) |
| Test Cases | `test-cases` | `D93F0B` (橙色) |
